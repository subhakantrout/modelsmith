ModelSmith — Full Design Proposal

A visual desktop app to uncensor, enhance, and compress any local AI model

Tech Stack

Frontend

  • Tauri v2 + React + TypeScript
  • React Flow (node-based pipeline editor)
  • Tailwind CSS + shadcn/ui
  • Recharts for data viz

Backend (Python Sidecar)

  • FastAPI server (Tauri sidecar)
  • PyTorch + transformers
  • Heretic/Abliterix (abliteration)
  • mergekit (model merging)
  • PEFT (LoRA injection)
  • llama.cpp (GGUF quant/compress)
  • bitsandbytes (4-bit loading)

Distribution

  • Single .exe/.dmg/.AppImage
  • Auto Python bundling
  • Optional: pip model backends
  • Windows/Mac/Linux
  • 4-8 GB RAM minimum

Pipeline Node Types

🧹 Abliterate — Remove censorship via directional ablation
🧬 Merge — Combine with other models (TIES, SLERP, DARE)
📦 Inject LoRA — Add fine-tuned skills without full merge
⚡ Compress — Quantize (GGUF), prune, or distill
🔬 Analyze — Scan capabilities, refusal score, layer importance
💬 Test — Chat with intermediate results at any pipeline stage
📤 Export — Save as GGUF, SafeTensors, or push to HF Hub
